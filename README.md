# deepnlp

1. 환경 셋팅

기본 전제> git과 anaconda가 설치되어 있어야 한다. 

```shell
(저장소 cloning)
$ git clone https://github.com/psygrammer/deepnlp.git

(소스 루트 디렉토리로 이동 후)
$ cd deepnlp

(실습용 아나콘다 환경셋팅을 위해 다음 명령 실행)
$ conda env create -f environment.yml
```

2. 실습환경

```shell
위의 환경셋팅이 끝나면 
anacoda navigator에서 deepnlp 가상환경을 찾아 jupyter notebook 실행
```
